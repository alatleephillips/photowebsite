Files Structure

Below are all the files included with the template:

  1. CSS
    • reset.css -> used for reseting default browser styles
    • style.css -> Main stylesheet.
  2. images -> contains all the images used in the template.
  3. includes
    • entypo -> "Entypo" font icon files.
    • fontawesome -> "FontAwesome" font icon files.
    • icomoon -> "IcoMoon" font icon files.
    • font -> "Playfairdisplay" is the font used in template.
  4. js
    • jquery.js

    • swiper.js

  5. And the html files.

HTML Structure

Template grid system is based on a 1100px width, and fullscreen layouts are enabled too.

For responsiveness the grid will support a tablet layout and a mobile layout

  • Between 1280px and 769px wide, the tablet layout will be enabled.
  • Below 769px wide, the mobile layout will be enabled.

JavaScript Structure

All the plugins needed in this template have been listed in files sturcture. The main template JavaScript file is "scripts.js", down below is the structure of the file:

  • Global section -> contains common functions used globally all over the pages.
  • Navigation Section -> contains all the functions used in menu.
  • Events Section -> contains all the functions used for events.
  • Main Functions Section -> contains all the functions used in drawing the layouts.
